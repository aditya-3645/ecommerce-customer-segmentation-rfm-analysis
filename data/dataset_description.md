\# Olist Brazilian E-Commerce Dataset



\## Dataset Overview



This project uses the Olist Brazilian E-Commerce Dataset containing information about customers, orders, payments, products, sellers, and reviews from a Brazilian marketplace.



\## Tables Used



\### customers



Contains customer information and location details.



\### orders



Contains order status and purchase timestamps.



\### order\_items



Contains product-level order information.



\### order\_payments



Contains payment values and payment methods.



\### order\_reviews



Contains customer review information.



\### products



Contains product metadata and categories.



\### sellers



Contains seller information and locations.



\## Project Goal



Perform RFM (Recency, Frequency, Monetary) analysis to segment customers into different business categories and identify opportunities for retention, upselling, and revenue growth.
