# E-Commerce Customer Segmentation \& RFM Analysis



## Project Overview



This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis on the Olist Brazilian E-Commerce Dataset.



The objective is to identify high-value customers, at-risk customers, and business growth opportunities using SQL, Python, and Power BI.



---



## Tech Stack



* MySQL

* Python

* Power BI

* SQL

* Pandas



---



## Dataset



Source: Olist Brazilian E-Commerce Dataset



The dataset contains information about:



* Customers

* Orders

* Order Items

* Payments

* Reviews

* Products

* Sellers



---



## Project Workflow



1. Data Import into MySQL

2. Data Validation

3. Exploratory Data Analysis

4. Customer Transaction Aggregation

5. RFM Metric Calculation

6. Customer Segmentation

7. Business Insights Generation

8. Power BI Dashboard Creation



---



## RFM Metrics



### Recency



Days since a customer's last purchase.



### Frequency



Total number of orders placed by a customer.



### Monetary



Total amount spent by a customer.
